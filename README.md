# 华南师范大学计算机学院项目团队知识库

- 文档 Markdown 根目录： `./docs`
- 基于 [VitePress](https://vitepress.dev/zh/) 生成

## Quick Start

项目运行环境：

- `Node.js v18` 以上
- `pnpm 8.15.0`

### 依赖安装

```bash
pnpm install
```

### 本地启动热更新服务器

```bash
pnpm dev
```

访问 [http://localhost:5173/wiki/](http://localhost:5173/wiki/) 即可。

### 打包构建

```bash
pnpm build
```
