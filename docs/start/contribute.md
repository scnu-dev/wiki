## 如何参与贡献

1. `scnu-dev` 团队成员直接在文档仓库中提交变更；
2. 其他同学 `fork` [本仓库](https://github.com/scnu-dev/wiki)，修订后向本仓库提出 **Pull Request**。

:::warning 须知
commit 信息请遵循 [Conventional Commits 规范](https://www.conventionalcommits.org/zh-hans/v1.0.0/)
:::
